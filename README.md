## Roadmap

- [ ] multistream-select 1.0
- [X] multiformats: [multiaddr](https://github.com/multiformats/multiaddr)

-  - [ ] Support multiaddr certhash
- [ ] crypto (RSA, ed25519, secp256k1)

- [ ] noise security protocol
- [ ] MDNS
- [ ] Gossip 1.1 pubsub
